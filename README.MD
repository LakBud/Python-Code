# Informasjonsteknologi 2 / Information Technology 2 Python Course 2025/2026

---

## Description:

- IT-2 is an college subject in Norway which in this case features Python.
- A showcase of all the resources and tests within IT-2
- This repo includes the 2025/2026 version of this subject.

---

## File System:

<!-- prettier-ignore -->
```
Informasjonsteknologi-2-python-course/
├── data/
│   └── json_files/ → Datasets used in tasks and syntax folder  
├── src/
│   ├── syntax/
│   │   └── school_tests/
│   │       └── syntax_files/ → Files made while syntax before each test 
│   └── tasks/
│       └── school_tests/
│           └── task_files/ → Assignments, exercises, and test practice
```

- The numbers on the files "syntax" and "tasks" means the files were created before a college python test.
- So all files on folder "1_before_first_test" means that the files were created before the first college test, etc.
- WARNING: Some files in the tasks-folder is written in Norwegian
